# Designing an exocortex
A hypothetical artificial information-processing system that would augment a brain's biological cognitive processes.[1] To me that means all the tools one uses to stay productive and get work done.

We create a lot of data during our day-to-day routines and it helps to have a well thought out system to organize it all.

The three main components of my system are tasks, notes, and files. Tasks are for helping keep track of steps I need to complete throught the day. Notes are for storing ideas and collecting information. Files are for projects, documents, and others.

For managing tasks, I use Things only on iOS.

For managing notes, I use a folder of markdown files that I edit with IAWriter on iOS and a text editor on my laptop.

For managing files, I use iCloud and the files app on iOS, and finder on my laptop.
