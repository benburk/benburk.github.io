This is an annotation of my [`macos.sh`](https://github.com/benburk/dotfiles/blob/master/dotfiles/bootstrap/macos.sh) script used to bootstrap new macOS installs.

## Install a fresh copy of macOS
1. Boot into recovery mode (hold `cmd+r` during reboot)
2. Format the boot drive and install macOS from scratch

## Command line tools
The Command Line Tools package gives you access to development tools for macOS. It includes `clang`, `gcc`, `git`, and `make` among others. You can view all of them by browsing `/Library/Developer/CommandLineTools`. Install them with:
```bash
sudo xcode-select --install
```

## Homebrew
[Homebrew](https://brew.sh/) is a package manager for macOS.
```bash
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

## Fish shell
A great alternative to bash with useful defaults.
```bash
brew install fish
chsh -s $(which fish)
```

## Some useful developer tools
```bash
brew install fzf rg fd git
```

## Settings
Configure system preferences using defaults.

Manual:
- finder -> new window shows icloud
- safari -> show status bar
- dock -> don't show recent apps
- trackpad -> secondary click -> click in bottom right corner
- Keyboard -> Shortcuts -> Mission Control -> ^n to switch to desktop n
- Keyboard -> Dictation -> Shortcut -> Right command key twice
- Accessibility -> Zoom -> Use scroll gesture with modifier keys


## Manual settings
**Safari**
- Extensions: bitwarden, IINA, vimari, wipr

**Terminal**
- Install Nord theme
- Profile -> Keyboard -> Use option as meta key
- Profile -> Uncheck scroll alternate screen
